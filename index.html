Evolución del Sistema (fecha aproximada de inicio enero 2025)

Fase 1: Arquitectura del sistema (Mes 1)

  • • Diseñar la arquitectura del sistema (frontend, backend y bases de datos)
  • • Especificar requerimientos funcionales
  • • Crear una interfaz para diseñar formularios

Fase 2: Sistema de generación de formularios (Mes 2-3)

  • • Desarrollo del constructor de formularios
  • • Estructura de base de datos para guardar formularios y sus respuestas

Fase 3: Sistema de Gestión de flujos (Mes 3-4)

  • • Diseñar una interfaz para configurar pasos del flujo
  • • Integración con el sistema de generación de formularios
  • • Seguimiento del estado de los flujos

Fase 4: Sistema Gestor documental (Mes 4-4.5)

  • • Integrar funcionalidad para asociar documentos a cada trámite o flujo
  • • Configurar roles y permisos para diferentes niveles de usuarios .
  • • Realizar pruebas del flujo de extremo a extremo con casos simples.
